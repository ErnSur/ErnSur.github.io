Editor Scripting Unity

Unity Editor Scripting Cheat Sheet

16.07.2022
3 minute read

Asset Management

  • AssetDatabase CRUD of Unity Editor
  • AssetPostprocessor & AssetModificationProcessor Take action after asset import
  • PrefabUtility Inspect / Modify Prefabs
    • Instantiate prefabs in the scene as “prefab instances” and not regular game objects
    • Create prefab variants
    • Add components
    • Apply overrides
    • Supports Undo out of the box

UI

  • Entry points:

    • MenuItem Create context menu items
    • Editor Create custom “Inspector view” for specific component (MonoBehaviour object)
    • EditorWindow Create custom windows
    • SettingsProvider Create custom view in Preferences or ProjectSettings windows
    • PropertyDrawer Customize input field for your serializable c# type
    • EditorApplication.projectWindowItemOnGUI Customize project window items
    • EditorApplication.hierarchyWindowItemOnGUI Customize hierarchy window items

  • Draw GUI

    • IMGUI
      • Static classes: GUI, GUILayout, EditorGUI, EditorGUILayout
    • UI Toolkit
      • Namespaces UnityEngine.UIElements and UnityEditor.UIElements

Scene View

  • Gizmos Draw things like tool handles or bounding boxes
  • Overlays.Overlay Panels like tool options or data

Serialization

  • JsonUtility
  • SerializedObject & SerializedProperty When you want to modify component/asset avoid modifying it directly, instead use this API to properly save assets, scenes, support Undo etc.
  • Undo When modifying scene objects or assets use this class to support undo feature.

Native Popups

  • EditorUtility.DisplayDialog Get confirmation from user
  • EditorUtility.DisplayPopupMenu Open context menu
  • Make user select path
    • EditorUtility.OpenFilePanel
    • EditorUtility.OpenFilePanelWithFilters
    • EditorUtility.OpenFolderPanel
    • EditorUtility.SaveFilePanel
    • EditorUtility.SaveFilePanelInProject
    • EditorUtility.SaveFolderPanel
  • Progress bar
    • EditorUtility.DisplayProgressBar
    • EditorUtility.ClearProgressBar

Advanced / Tricks

  • IMGUI.Controls.TreeView IMGUI TreeView
  • IMGUI.Controls.AdvancedDropdown Dropdown with pages as in “Add Component” button.
  • EditorGUIUtility.IconContent Load icon texture from editor resources. More at Editor Icon Browser
  • Open About Unity menu and type “internal” to toggle editors developer mode.
    • Reload and inspect editor window from its tab context menu
    • Enable “Debug-Internal” inspector mode from tab context menu
    • Bunch of other internal tools, you can look for those in Unity C#Reference repo

Resources

  • Unity C#Reference repo
  • My GitHub projects & Stars
  • Unity Forums

using UnityEditor;
using UnityEditorInternal;
using UnityEngine;

internal static class ContextMenuItems
{
    private const string RevertPrefabTransformMenuName = "CONTEXT/RectTransform/Revert Prefab Transform";

    [MenuItem("GameObject/Copy Transform #^C")]
    public static void CopyTransform(MenuCommand command)
    {
        var obj = command.context;
        if (obj == null)
            obj = Selection.activeGameObject;
        if (obj is GameObject go)
        {
            ComponentUtility.CopyComponent(go.transform);
            var sceneView = SceneView.lastActiveSceneView;
            if (sceneView != null)
                sceneView.ShowNotification(new GUIContent($"Copied {go.name} Transform"));
        }
    }

    [MenuItem("GameObject/Paste Transform #^V")]
    public static void PasteTransform(MenuCommand command)
    {
        var obj = command.context;
        if (obj == null)
            obj = Selection.activeGameObject;
        if (obj is GameObject go)
        {
            ComponentUtility.PasteComponentValues(go.transform);
            var sceneView = SceneView.lastActiveSceneView;
            if (sceneView != null)
                sceneView.ShowNotification(new GUIContent($"Paste {go.name} Transform"));
        }
    }

    [MenuItem(RevertPrefabTransformMenuName)]
    private static void RevertPrefabTransform(MenuCommand command)
    {
        var serObj = new SerializedObject(command.context);
        var prop = serObj.GetIterator();
        while (prop.NextVisible(true))
        {
            PrefabUtility.RevertPropertyOverride(prop, InteractionMode.UserAction);
        }
    }

    [MenuItem(RevertPrefabTransformMenuName, validate = true)]
    private static bool RevertPrefabTransformValidate(MenuCommand command)
    {
        var obj = command.context;
        return PrefabUtility.IsPartOfPrefabInstance(obj);
    }
}
